Habilidades tecnológicas

+

HTML 5

Sus siglas vienen del inglés 'HyperText Markup Language'. Traducido al español quiere decir 'Lenguaje de Marcado e HiperTexto'. Este lenguaje de marcado nos permite crear la estructura (o cuerpo) básica de un sitio web mediante tags (etiquetas). El número cinco (5) hace referencia a la versión en la que se encuentra actualmente. Todos los titulos, párrafos, formularios, videos, entre muchísimas cosas más, las tenemos gracias a este maravilloso lenguaje.

+

CSS

CSS es un lenguaje de estilo. Sus siglas también tienen un significado 'Cascading StyleSheet', su traducción es 'Hojas de estilo de cascada'. Seguro ya sabrás qué quiere decir el número tres (3), ¿verdad? ¡Exacto! Su versión actual. Este lenguaje de estilos nos permite... pues, darle estilos al cuerpo de nuestro sitio web. Es el maquillaje de HTML. Todo el diseño de un sitio va directamente relacionado con este lenguaje.

+

JavaScript

¡Bendito JavaScript! Es un lenguaje de programación que nos permite añadir, eliminar, aplicar animaciones o efectos a los componentes de nuestro sitio. Es la herramienta perfecta para realizar sitios web dinámicos y muy atractivos de la mano de CSS y HTML. Con el paso del tiempo ha evolucionado de manera exponencial y ahora mismo es absolutamente necesario conocerlo si quieres ser desarrollador web.

+

Bootstrap

Es una de las estructuras de trabajo más usados en el mundo. Este framework hace mucho más sencillo el diseño y posicionamiento de los elementos con CSS. Te ahorras mucho tiempo gracias a que por defecto él trae las herramientas o código interno para que tú no tengas que escribir nada más que clases específicas que tiene predefinidas. Con una de estas clases de bootstrap te podrías ahorrar 5 o más lineas de código en CSS.

+

Cakephp

CakePHP es un framework de código abierto basado en PHP diseñado para facilitar el desarrollo de aplicaciones web de manera eficiente y estructurada. Utilizando la arquitectura MVC (Modelo-Vista-Controlador), CakePHP permite a los desarrolladores crear aplicaciones web de forma rápida y robusta. Con CakePHP, los desarrolladores pueden aprovechar una amplia gama de características y funcionalidades predefinidas, lo que les permite ahorrar tiempo y esfuerzo en la creación de aplicaciones desde cero.

+

Laravel

Laravel es un framework de código abierto de PHP utilizado para desarrollar aplicaciones web. En términos más sencillos, es una herramienta que permite a los desarrolladores crear aplicaciones web de forma más rápida y eficiente, gracias a su arquitectura MVC (Modelo-Vista-Controlador) y su amplia variedad de características y funcionalidades predefinidas.

+

Cisco

Cisco es una empresa líder en tecnología de redes y comunicaciones. Ofrece una amplia gama de productos y servicios, incluyendo routers, switches, firewalls y soluciones de networking. Como ingeniero en sistemas computacionales, comprender los productos y tecnologías de Cisco es fundamental para diseñar, implementar y mantener infraestructuras de redes seguras y eficientes. Los conocimientos en tecnologías de Cisco son altamente valorados en el campo de la informática y pueden abrir muchas oportunidades profesionales en el ámbito de las redes y las telecomunicaciones.

+

PHP

PHP es un lenguaje de programación de uso general ampliamente utilizado para el desarrollo de aplicaciones web dinámicas. Con una sintaxis sencilla y flexible, PHP se integra fácilmente con HTML, lo que lo convierte en una herramienta poderosa para la creación de sitios web interactivos y funcionales. Como ingeniero en sistemas computacionales, el dominio de PHP te permite crear aplicaciones web robustas y escalables. Con PHP, puedes trabajar con bases de datos, gestionar sesiones de usuarios, crear formularios dinámicos y mucho más. Además, PHP cuenta con una amplia comunidad de desarrolladores y una gran cantidad de recursos disponibles.

+

phpMyAdmin

phpMyAdmin es una herramienta de administración de bases de datos MySQL basada en web, escrita en PHP. Proporciona una interfaz gráfica intuitiva que permite a los usuarios administrar sus bases de datos MySQL a través de un navegador web. Como ingeniero en sistemas computacionales, el dominio de phpMyAdmin es fundamental para la gestión eficiente de bases de datos MySQL. Con phpMyAdmin, puedes crear, eliminar, modificar y gestionar tablas, campos, índices y usuarios de bases de datos MySQL de forma fácil y segura. Además, phpMyAdmin ofrece características como la importación y exportación de datos, la ejecución de consultas SQL.
